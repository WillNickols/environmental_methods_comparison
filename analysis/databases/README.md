# Databases

- The folder `figures/` contains upset plots of the taxa in the standardized databases
- The folder `ncbi_taxdump/` contains the nodes and names dmp files downloaded from [NCBI](https://ftp.ncbi.nlm.nih.gov/pub/taxonomy/)
- The folder `raw_databases/` contains the original file downloaded as the database for each taxonomic assignment tool
- The folder `scripts/` contains scripts to standardize the raw databases and create upset plots
- The folder `standardized_databases/` contains standardized versions of each database, mapping the taxa as it would show up in a sample to its standardized IDs
- The folder `taxa_lists/` shows an aligned list of taxa across the tools at each taxonomic level
