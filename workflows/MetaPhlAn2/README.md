# MetaPhlAn 2 workflow

This folder contains code for the MetaPhlAn 2 workflow.

# Versioning:

- MetaPhlAn2: 2.6.0
  - Database: mpa_v20_m200

# Running

Commands of the following form were run:

```
python mpa2_workflow.py -i kneaddata_cleaned \
  -o metaphlan2_output \
  --input-extension fastq.gz \
  --paired paired
```
