# Workflows

This directory contains the workflows that were run for this project.  The folders [GTDB-Tk](GTDB-Tk), [Kraken2_Bracken](Kraken2_Bracken), [MetaPhlAn 3](MetaPhlAn 3), [MetaPhlAn 4](MetaPhlAn 4), [MetaPhlAn2](MetaPhlAn2), [Metaxa2](Metaxa2), [assembly_workflow_MEGAHIT](assembly_workflow_MEGAHIT), [assembly_workflow_metaSPADES](assembly_workflow_metaSPADES), and [mOTUs3](mOTUs3) contain workflows for the taxonomic profiling tools evaluated.  The folder [Pre-processing](Pre-processing) contains scripts to download and clean read files.  The folder [Simka](Simka) contains a workflow to run Simka.  The folder [Simulations](Simulations) contains the workflow to generate taxonomic profiles and reads for synthetic evaluation.  Each folder contains its own README describing its workflow.
